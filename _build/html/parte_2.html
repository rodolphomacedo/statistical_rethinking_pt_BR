Defini√ß√£o de W:¬∂

  • √â o n√∫mero relativos (proporcionais) de maneiras na quais podemos ver W, dado \(N\) e \(p\).

  • Objetivo: Uma fun√ß√£o matem√°tica que atribui um n√∫mero relativo de maneiras pelas quais veremos o valor espec√≠fico de W, sem que precisarmos de construir todas as infinitas contagens relativas manualmente.

Para come√ßar, vamos supor que coletamos 3 amostras:

\[[W, L, W]\]

Qual √© o n√∫mero relativo (ou seja, proporcional) de maneiras de vermos essa sequ√™ncia de amostras, dado algum valor de \(p\).

N√£o sabemos qual √© o valor de \(p\), iremos dar qualquer valor de \(p\) que esteja entre \(0\) e \(1\).

Multiplicaremos todas as propor√ß√µes, pois os dados s√£o independentes, pela teoria temos que probabilidades independentes s√£o multiplicadas pela regra do produto.

E por que multiplicamos? Multiplicamos pois √© a maneira r√°pida de contar todas as possibilidades pela bifurca√ß√£o que o dados podem fazer. Nada mais √© um modo mais r√°pido de contar todas as possibilidades:

Distribui√ß√£o de W (Verossimilhan√ßas)¬∂

E nessa f√≥rmula, se extrapolarmos a para uma sequ√™ncia de qualquer comprimento (\(N\)), chegaremos a conhecida Distribui√ß√£o Binomial.

Probabilidade a posteriori¬∂

  • A estimativa bayesiana √© sempre uma distribui√ß√£o sobre os par√¢metros, \(P\{ parametros | dados\}\):

\[ P\{p | W, N \} \]

O c√°lculo via o teorema de bayes:

\[ P\{ p | W, N \} = \frac{ P\{W | N, p\} \times P\{ p \} } {\sum P\{W | N, p \} P\{p\}, \forall p } \]

onde (\(\forall p\)) significa para todo os \(p\)‚Äôs.

\[ \mbox{Posteriori} = \frac{\mbox{Probabilidade das vari√°veis observadas } \times \mbox{Priori}} {\mbox{Constante de Normaliza√ß√£o}} \]

C√°lculo da Posteriori¬∂

N√£o temos escolha a n√£o ser calcular a posteriori, por√©m podemos calcular de algumas formar diferentes:

  • Abordagem Anal√≠tica (Geralmente imposs√≠vel)

  • Via grade de aproxima√ß√£o (Intenso computacionalmente)

  • Aproxima√ß√£o Quadr√°tica - Tamb√©m conhecida com Aproxima√ß√£o de Laplace (√â limitada)

  • Monte Carlo via Markov Chain MCMC (Intenso Computacionamente)

Via Grade de Aproxima√ß√£o (Grid Approximation)¬∂

  • A probabilidade √° posteriori √© o produto normalizado das:

    • probabilidade dos dados

    • probabilidade √° priori

  • A grade usa um n√∫mero finito de pontos ao inv√©s usar o espa√ßo cont√≠nuo.

  • Essa abordagem √© muito custosa quando temos multiplos par√¢metros.

Resumindo as tarefas:

  • Quanto da probabilidade a posteriori abaixo, acima ou entre valores espec√≠ficos do par√¢metro?

N√≥s que decidimos qual usar!

Existem, em estat√≠stica, basicamente dois tipos de intervalos: Intervalo de Percentil e HPDI (Highest posterior density intervals - O maior intervalo da densidade posteriori)

Intervalos de Percentil: √Åreas s√£o em cada regi√£o (bons para serem usados em distribui√ß√µes sim√©tricas)

HPDI:O intervalo mais estreito poss√≠vel que cont√©m a massa de probabilidade.

O que precisamos saber √© que o um intervalo √© apenas um resumo, eles comprimem a informa√ß√£o, mas o que geralmente importa √© o formato da posteriori, n√£o h√° nada nos limites que geram o intervalos que tenha propriedades m√°gicas.

# Fun√ß√£o que retorna a quantidade de pontos proporcional dentre um range
def probability_posteriori(hipoteses, posteriori, param_sup, param_inf=False, qtd_amostra=1000, plot=True):
    
    # Calculando o tamanho da amostra da posteriori
    amostra = np.random.choice(p_s, size=qtd_amostra, replace=True, p=posteriori)
    
    # Quantidade de pontos da amostra da posteriori que s√£o maiores do que o par√¢metro_1 
    prob = amostra <= param_sup
    
    # Se o par√¢metro inferiori foi passado.
    if param_inf and param_inf < param_sup:
        prob_param_inf = amostra >= param_inf
        prob = prob * prob_param_inf  # Juntando ambas contagens do param_1 e do param_2
        
    # Configura√ß√µes do plot
    if plot:
        hipoteses_filtrado_superior = hipoteses <= param_sup  # Todas as hip√≥teses que s√£o menores que param_sup
        
        # Se o par√¢metro inferiori foi passado.
        if param_inf:
            hipoteses_filtrado_inferior = hipoteses >= param_inf
            posteriori_filtrada = posteriori * hipoteses_filtrado_superior * hipoteses_filtrado_inferior
            texto = ('Probabilidade da propor√ß√£o de √°gua (p) na superf√≠cie da Terra estar entre ' + str(param_inf) + ' e ' + str(param_sup) + ' √© igual √† ' + str(round(np.sum(prob) / len(amostra) * 100)) + '%, dado a amostra.')
        else:
            posteriori_filtrada = posteriori * hipoteses_filtrado_superior
            texto = ('Probabilidade da propor√ß√£o de √°gua (p) na superf√≠cie da Terra estar entre 0 e ' + str(param_sup) + ' √© igual √† ' + str(round(np.sum(prob) / len(amostra) * 100)) + '%, dado a amostra.')
             
        plt.figure(figsize=(17, 9))
        plt.plot(hipoteses, posteriori)  # Plot posteriori
        plt.fill_between(hipoteses, 0, posteriori_filtrada, alpha=0.5)
        
        plt.title('Probabilidade das hip√≥teses da superf√≠cie da Terra estar coberta por √°gua')
        plt.xlabel('Hip√≥teses, $p_s$')
        plt.ylabel('Posteriori')
        plt.grid(axis='x', ls='--', color='gray')

        plt.show()
        
        print(texto)
        
    else:
        
        return np.sum(prob) / len(amostra)
